[檔案說明及使用方法]

GA交易策略.py:本研究GA訓練及測試代碼
---------------------------------------------------------------------
1.打開CMD
2.輸入python GA交易策略.py
3.程式將會開始訓練並進行模擬交易，待程式執行完畢後即完成一次實驗
4.實驗結果將產生再同目錄下，檔名:GA交易策略.txt 

PS:本研究GA獲利是以30次實驗平均值作為交易策略的整體績效
PS:理想交易策略實驗也是透過此代碼實現，差別僅在於測試期及訓練期需調整為同一時間。
---------------------------------------------------------------------

常用交易策略(以K為例):
---------------------------------------------------------------------
1.打開CMD
2.輸入python 常用交易策略-K指標.py
3.實驗結果將產生再同目錄下，檔名:常用交易策略-K指標.txt
